### 上手指南
    需要一定的硬件配置以及编程基础



###### 开发前的环境配置
1. IDEA
2. Node.js v18.18.2
3. npm 10.2.1



###### **搭建步骤**
1. Vite搭建Vue+TypeScript环境
2. 进入到根目录执行npm install下载所有所需的依赖

<br /><br />
